# Splinter

This extension analyzes and displays the ORM operations in the code.

## Features

Supported ORMs:

- [TypeORM](https://anonymous.4open.science/r/splinter-eslint)
- [Django ORM](https://anonymous.4open.science/r/splinter-mypy)
